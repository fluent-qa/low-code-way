# Low Code Software QA

Just want to share some real experiences in startup company.
This is for QA who struggling in a small startup company to build some tools to help increase productivity.
Because of the limitation of the budget, leverage some open-source tools is the only way to achieve it.

Let's say, How to Be Less struggling for QA in small startup company?

In startup company, how to solve problem with less cost?

1. what's kind of problem you want to solve?
2. Is there any existing open-source tool to solve it?
3. How to use the open-source tool?
4. How to integrate different tools
