# AVO demo

[avo](https://github.com/avo-hq) is a ruby on rails project.

## Installation
- install rvm
```shell
curl -sSL https://get.rvm.io | bash -s stable
```
- install ruby
```shell
rvm install 3.1.1
ruby use 3.1.1 --default
```
- install ruby on rails and create demo project
```shell
rails new  qa-adminss
```
- install avo
